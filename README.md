# Getting Started

Use Python3 interpreter

## Install packages

### `pip install flask flask_cors h5py numpy`

## Run server

### `python server.py`
Server will run on
Open [http://0.0.0.0:4567/api/](http://0.0.0.0:4567/)

## Endpoints

### `/api/measurement-separate`
### `/api/measurement-all`
### `/api/glucose`
